Disc Restock Bot

A website designed for disc golfers looking to track the availability of almost any disc on the market.

How it works:

  1. Type the name of the disc model and brand in the search bar. Keep in mind that the website is not complete, so some brands or models may not appear.
  2. Click the corresponding model that appears.
  3. Wait for the application to search for stock on select disc vendors.
  4. Follow the instructions that appear on screen. The application will either tell you the websites that have the disc in stock, or it will tell you that no stock was found. From here, the application will guide you through placing a tracker.
